Dasar

Container

Kotak serbaguna untuk menata, memberi dekorasi, dan mengatur jarak.

Container(
  height: 80,
  width: 80,
  color: Colors.blue,
)

Text

Menampilkan string teks dengan berbagai gaya.

Text(
  'Halo Dunia!',
  style: TextStyle(fontSize: 18),
)

Menampilkan ikon grafis dari set Material Design.

Input & Seleksi

ElevatedButton

Tombol standar dengan bayangan yang merespon klik pengguna.

ElevatedButton(
  onPressed: () {},
  child: Text('Klik'),
)

TextField

Kolom input teks yang memungkinkan pengguna mengetik menggunakan keyboard.

TextField(
  decoration: InputDecoration(
    labelText: 'Nama Anda',
  ),
)

Switch

Tombol on/off untuk mengubah satu pengaturan.

Switch(
  value: true,
  onChanged: (val) {},
)

Checkbox

Kotak centang untuk memilih satu atau beberapa opsi dari sebuah set.

Checkbox(
  value: true,
  onChanged: (val) {},
)

Menampilkan Info

Image

Menampilkan gambar dari berbagai sumber (network, asset, file).

Image.network(
  'https://placehold.co/400'
)

CircleAvatar

Sebuah lingkaran yang biasanya digunakan untuk menampilkan gambar profil atau inisial.

CircleAvatar(
  backgroundImage: NetworkImage(..),
  radius: 30,
)
KM

Chip

Elemen ringkas yang merepresentasikan input, atribut, atau aksi. Sering digunakan untuk filter atau tag.

Chip(
  label: Text('Tag Populer'),
  backgroundColor: Colors.amber,
)
Tag Populer

LinearProgressIndicator

Menunjukkan progress dalam bar horizontal. Bisa tak terbatas (indeterminate) atau berdasarkan nilai.

LinearProgressIndicator(
  value: 0.7, // 70%
)
