Katalog Widget Dasar Flutter

Referensi cepat untuk memahami blok bangunan fundamental dalam Flutter. Semua yang Anda lihat di aplikasi Flutter adalah Widget.

Container

Sebuah "kotak" serbaguna yang bisa diatur ukuran, warna, padding, dan dekorasinya. Fondasi untuk banyak layout.

Container(
  padding: EdgeInsets.all(16),
  color: Colors.teal[300],
  child: Text('Box'),
)
Box

Text

Widget paling esensial untuk menampilkan tulisan. Gayanya bisa diubah sesuai kebutuhan.

Text(
  'Halo, Flutter!',
  style: TextStyle(
    fontSize: 20,
    fontWeight: FontWeight.bold,
  ),
)

Halo, Flutter!

Column & Row

Menyusun widget-widget lain dalam barisan vertikal (Column) atau horizontal (Row).

// Menyusun ke bawah
Column(
  children: [
    Text('Atas'),
    Text('Bawah'),
  ],
)
Atas
Bawah

ElevatedButton

Tombol standar dengan efek terangkat (shadow) yang merespon ketika ditekan oleh pengguna.

ElevatedButton(
  onPressed: () {
    // Aksi ketika ditekan
  },
  child: Text('Tekan Saya'),
)

Icon

Menampilkan sebuah gambar ikon dari koleksi Material Design yang sudah disediakan oleh Flutter.

Icon(
  Icons.favorite,
  color: Colors.pink,
  size: 48,
)
♥

Stack

Memungkinkan Anda untuk menumpuk widget di atas widget lainnya. Sangat berguna untuk layout kompleks.

Stack(
  children: [
    Container(width: 100, height: 100, color: Colors.red),
    Text('Tumpukan'),
  ],
)
Tumpukan
